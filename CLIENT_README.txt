=====================================================
      DASHBOARD ITA - INSTRUCTIONS UTILISATEUR
=====================================================

Pour lancer le dashboard :

1. Double-cliquez simplement sur "ITA Dashboard.exe"

2. Un terminal s'ouvrira (NE PAS LE FERMER)

3. Le dashboard s'ouvrira automatiquement dans votre 
   navigateur web après quelques secondes

4. Utilisez le dashboard normalement dans votre navigateur

5. Quand vous avez terminé, fermez la fenêtre du terminal
   pour arrêter complètement l'application

=====================================================
          PROBLÈMES COURANTS
=====================================================

Si vous rencontrez une alerte de sécurité Windows :
- Cliquez sur "Informations complémentaires"
- Puis sur "Exécuter quand même"

Si le dashboard ne s'ouvre pas automatiquement :
- Ouvrez votre navigateur et allez à http://localhost:8502

Si vous obtenez un message "Pas de connexion" dans le dashboard :
- Vérifiez que vous avez accès au serveur SQL (192.168.1.250)
- Vérifiez votre connexion au réseau d'entreprise
